# Skills 技能库总索引

本目录包含 3 个技能包，用于辅助 AI 构建高质量网站。技能按**渐进式披露**加载：AI 先读触发条件决定是否调用，调用后再读完整内容决定怎么执行。

## 技能包清单

| 技能包 | 触发条件 | 核心能力 |
|--------|---------|---------|
| **[style-skills](style-skills/index.md)** | 需要独特视觉风格、避免 AI 模板感 | 23+ 前端设计技能：配色、排版、动效、布局、打磨 |
| **[ui-skill](ui-skill/index.md)** | 需要配色方案、字体搭配、设计系统推荐 | BM25 搜索引擎 + 15 个知识库 + 智能设计系统生成 |
| **[storytelling-for-ux](storytelling-for-user-experience-crafting-stories-for-better-design-skill.pdf/index.md)** | 网站需要叙事性内容（品牌故事、案例、介绍） | 28 个叙事技能：故事收集、构建、表达、协作、伦理 |

## 按场景推荐

| 构建场景 | 推荐技能 |
|---------|---------|
| 全新网站设计 | ui-skill（生成设计系统）→ style-skills/frontend-design（设计方向） |
| 优化已有设计 | style-skills/audit（审查）→ 针对性调用 colorize/typeset/arrange |
| 品牌故事页面 | storytelling-for-ux（叙事构建）+ style-skills/editorial 方向 |
| 作品集展示 | ui-skill（配色+排版）→ style-skills/animate（动效）→ polish（打磨） |
| 落地页优化 | ui-skill/landing 搜索 → style-skills/bolder（视觉冲击）→ clarify（信息架构） |
