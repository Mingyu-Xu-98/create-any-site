# 跨学科故事应用与受众适配

## 何时使用
- 沟通对象包含管理者、专家、外部人员等多元角色
- 自身与受众存在经验、权力或文化差异
- 设计问题涉及多个社会、文化、法律或资源维度
- 需要在同一核心事件基础上向不同受众传达不同重点

## 执行步骤
1. **识别受众关系类型**：
   - 同行社区成员
   - 外部人士
   - 向上沟通（管理者、客户）
   - 专家身份传递知识
2. **评估共享背景程度**：
   - 高度共享：可使用专业术语和隐含知识
   - 低度共享：必须解释术语、避免假设共同认知
3. **动态切换叙事视角**：
   - 在宏观战略（50,000英尺）与具体细节（100英尺）间快速切换
   - 避免过度宏观（脱离现实）或过度微观（陷入泥潭）
4. **整合多视角**：
   - 识别关键视角（教育、监管、读写能力、资源等）
   - 尊重地共享不同视角，承认语境重叠与相互影响
   - 使用简单故事或例子推动新想法
5. **重新表述复杂概念**：
   - 将技术限制重新定义为用户可理解的问题
   - 保留足够中间细节确保逻辑连贯

## 输出标准
生成一个能动态适配多元受众关系的叙事策略，包括内容取舍、术语处理、视角切换节奏及共情验证机制，有效促进跨学科理解和协作。
