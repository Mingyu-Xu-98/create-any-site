# 索引条目结构与用途

## 何时使用
- 用户需在已分页的印刷文档中快速找到某个主题的位置
- 用户希望了解某主题在全书中的分布情况（多个页码或页码范围）
- 用户希望通过交叉引用发现相关概念
- 文档为线性结构且具备标准索引（不适用于无页码或纯网页式数字内容）

## 如何使用
1. **按字母顺序查找主条目**：索引条目按英文字母顺序（A–Z）排列，先定位目标关键词的首字母。
2. **识别条目类型**：确认条目是主题词、人物/著作、方法技术，还是叙事元素等。
3. **读取页码信息**：
   - 单页用逗号分隔（如 `47, 316`）
   - 连续页码用连字符表示（如 `121-122`）
4. **处理交叉引用**：若条目包含“See also”或类似提示，转至所列关键词继续检索。
5. **注意延续标记**：若看到“(continued)”，说明该主题内容跨多个非连续区域，需继续查找后续条目。
6. **综合多维度信息**：同一主条目下可能涵盖伦理、结构、方法等多个子维度，结合上下文理解其在书中的完整含义。

> 注意：仅适用于已分页、内容结构化的印刷或PDF类文档。不适用于无固定页码的动态网页或应用内帮助系统。
