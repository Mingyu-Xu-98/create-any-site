# 多模态演示设计

## 何时使用
- 正在为包含故事叙述的商业演示设计幻灯片
- 已构建完整的故事叙述和明确核心信息点
- 需提升观众多模态参与度，而非仅传递文字信息
- 不适用于纯数据报告或法律文件等需字面精确传达的场景

## 执行步骤
1. **确认故事完整性**：确保已拥有完整的故事情节后再开始制作幻灯片。
2. **采用互补原则**：幻灯片不应重复口头内容，而应提供与之互补的视觉元素。
3. **设计视觉内容**：
   - 使用富有表现力的图像（imagery）
   - 仅保留少量关键词以强化核心观点
4. **应用多模态认知原理**：利用观众同时通过视觉和听觉接收信息的特点，使两种模态提供不同但协同的信息。
5. **遵循核心理念**：“幻灯片不是演示本身”——演示是视觉、听觉、讲述者三者的综合体验。

## 输出标准
生成一组与口头叙述互补而非重复的视觉幻灯片，有效提升观众多模态参与度和信息吸收效果。
