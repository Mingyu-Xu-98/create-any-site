# UX叙事结构详解

## Now-Then-Now结构
- **Now_1**：以当下具体事件开篇（例：“现在我身高6英尺3英寸，选车意味着选择坐进去最不疼的那辆”）
- **Then**：转向相关过去经历（例：“10岁时我能轻松坐进任何车”）
- **Now_2**：回归当下但感知已变（例：“如今我不再驾驶Camaro，但我开的方盒子在我心里全是Camaro”）

## 处方式结构模板
```
标题：[明确主题]
Given [用户身份、环境、当前状态]
And [更多上下文...]
When [关键事件或触发点]
Then [解决方案或积极影响]
And [另一结果]
```

## 其他结构
- **英雄式**：参考约瑟夫·坎贝尔“英雄之旅”，常用于产品发布叙事
- **框式**：开头与结尾形式相同，形成闭环
- **层叠式**：通过叠加意象构建多维体验
- **情境插曲式**：插入物理/情感细节“岔路”增添维度
