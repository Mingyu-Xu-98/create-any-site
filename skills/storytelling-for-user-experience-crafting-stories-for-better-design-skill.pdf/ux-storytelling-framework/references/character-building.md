# 角色构建技巧

## 核心原则
- **基于真实数据**：角色应源自用户研究，除非刻意选择非典型角色以拓展理解
- **展示而非陈述**：用行为展现特质（例：“她每周为两个孩子计划四顿正餐，并巧妙延展为七顿”而非“她是精打细算的单亲妈妈”）
- **设置伏笔**：早期引入可后续呼应的特质（如癌症幸存者体现耐心）
- **利用原型**：使用“足球妈妈”等常见标签快速建立预期，注意文化差异
- **纳入多元视角**：自然融入经济、族裔、地理位置或残障等特征

## Dictio技巧
- 角色早期做出绝对化声明（如“我讨厌手机”）
- 后续行为与之矛盾，揭示“说”与“做”的张力
- 用于展示产品如何克服初始抗拒
