# 英国老式火车车门开启操作指南

## 适用场景
- 列车已停靠或减速至可安全下车速度
- 车厢为英国老式设计：无内部门把手，车窗可部分打开
- 不适用于现代全自动车门列车或车窗完全封闭的车厢

## 操作步骤
1. **提前准备**：列车接近目的地车站并开始减速时，立即起身前往最近的车门。
2. **确认门型**：检查车门内侧是否无把手。若整节车厢均无内部门把手，则属于典型英国老式设计。
3. **检查车窗**：确认相邻车窗可部分打开。若不可开启，立即寻找乘务员或移至其他车厢。
4. **操作外部门把手**：将手伸出窗外，抓住车门外侧的物理门把手，向外拉动或旋转以解锁并打开车门。
5. **安全下车**：仅在列车完全停止或极低速运行时开门；切勿在列车加速或高速运行时尝试跳车。
6. **迅速离开轨道区域**：下车后立即远离轨道，确保人身安全。

## 安全警示
- 此设计为“安全功能”，旨在防止运行中误开车门。
- 强行跳车可能导致严重人身伤害，绝对禁止。
- 若车窗损坏或无法打开，请勿强行操作，应寻求工作人员协助。
