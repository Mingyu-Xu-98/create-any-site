# 叙事结构与要素设计

## 何时使用
- 需要通过故事引发受众情感共鸣或增强记忆
- 故事包含人物、情境或事件
- 需要在缺乏显著动作时传达体验质感
- 需要向受众介绍陌生、复杂或令人不安的新概念

## 执行步骤
1. **确定基本叙事要素**：
   - 视角（Perspective）：故事被讲述的视点
   - 人物（Characters）：推动情节和引发共情的核心
   - 背景（Context）：时间、地点和社会文化设定
   - 意象（Imagery）：唤起感官或情感反应的描述
   - 语言（Language）：叙述风格及角色说话方式
2. **选择合适叙事结构**：
   - **熟悉到陌生**：从受众熟悉情境引入新概念
   - **上下文插叙**：通过感官细节传达体验质感
   - **结构与情节平衡**：结构为骨架，情节为事件呈现顺序
3. **优化意象使用**：
   - 提供适量物理或情感上下文
   - 避免过多意象压倒受众或过少导致空洞
   - 确保意象服务于故事核心目的
4. **应用上下文插叙**：
   - 作为开头引入背景
   - 作为结尾增添情感层次
   - 在框架式故事中对比起始与终态

## 输出标准
生成一个经过要素优化和结构设计的叙事段落，包含适量、相关且能激发受众想象的具体细节，有效传达核心观点并引发情感共鸣。
